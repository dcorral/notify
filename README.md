# Notify

`Notify` runs a command and plays a notification sound when it finishes.

## Requirements

- GCC
- SDL2

## Steps

- **Compile:** Run `make`.
- **Install:** Run `sudo make install` to install to `/usr/local/bin`.
- **Use:** Run `notify <your-command>`.
  Example: `notify ls -la`
- **Clean up:** Run `make clean`.
